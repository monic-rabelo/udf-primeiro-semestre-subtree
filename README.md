# Visão geral

Repositório criado para abrigar os materiais e atividades realizados no primeiro semestre, com intuito de organizar os conteúdos e centraliza-los em uma só pasta.
| Matéria | Conteúdos |
| ----------- | ----------- |
| algoritmos  | gerenciamento de alunos com struct, calculadora científica e insertion sort  |
| design-profissional| atividade básica testando o git, A1 com certificação, aplicação para vaga de estágio na área, currículo e portfólio com projeto de dados e calculadora científica  |
| front-end  | projeto02 desenvolvido em aula e projeto final utilizando CSS, HTML, JavaScript e frameworks.  |
| modelagem-banco-dados  | projeto final sobre o trabalho de "turismo" com modelo conceitual, lógico e físico do banco de dados, juntamente ao dicionário de dados e também aos prints sobre atualização, exclusão e inserção de dados no banco.|

---

## Matérias e contéudos presentes

### Algoritmos e pensamento computacional
- Códigos feito em **linguagem C**, onde possuem o uso de estruturas de condição, repetição, estrutura de dados e estudos gerais de algoritmos (algoritmos de ordenação).
- Feitos com o uso de IDEs, são elas: Visual Studio Code, Dev C++ e GDB online.
- Aplicação de lógica e pensamento computacional, entendendo conceitos de pensamento computacional: abstração, decomposição, reconhecimento de padrões e algoritmos.
- Capacidade de analisar logicamente problemas levantados em sala de aula, tal qual comentar o código, mostrando entedimento sobre o assunto.

### Design profissional
- Conteúdos sobre participação em eventos, carreira, currículo profissional e portfólio.
- Uso de ferramentas como: Word e LaTeX (Overleaf)
- Portfólio profissional com dois projetos relevantes: um externo à faculdade e outro desenvolvido dentro dela.
- Código de atividade aplicando conhecimentos sobre GitHub.

### Front-end
- Códigos desenvolvidos com as principais ferramentas para desenvolvimento web: **HTML, CSS e JavaScript**
- Aplicação de tags do HTML e suas estruturas básicas (head, body) e estilização com CSS.
- Uso de JavaScript para adicionar funcionalidade e interatividade nos sites.
- Uso de frameworks: Materialize e Bootstrap

### Modelagem de Banco de Dados
- Conteúdos sobre o projeto final de modelagem, com aplicação de **código SQL, diagrama ER e dicionário de dados**
- Projeto final feito com as aulas, ensinando sobre modelagem como princípio.
- Código SQL contendo insertions, updates e deletes.
- Tabelas, relacionamentos e cardinalidades aplicados no contexto do projeto sobre "Turismo"
